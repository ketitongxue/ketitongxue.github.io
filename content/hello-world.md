## 关于这个博客

这个博客使用 [Hugo](https://gohugo.io/) 搭建，主题使用简洁的 [Paper](https://github.com/nanxiaobei/hugo-paper)。

## 快速开始

Hugo 是一个快速的静态网站生成器，使用简单：

```bash
# 新建文章
hugo new content/post-title.md

# 本地预览
hugo server -D

# 构建网站
hugo
```
